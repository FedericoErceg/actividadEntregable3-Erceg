Que es nodeJS?
Es un entorno de desarollo completo sobre el cual viven y se ejecutan nuestros programas. Trabaja con el motor V8 que es el mismo de Chrome, el cual nos permite  convertir codigo de javascript a codigo maquina, para poder ser procesado correctamente.

const numArr = [];
for(let i = 0; i < 10000; i++){
    const numeroAleatorio = Math.floor(Math.random() * 20 + 1); //Math.floor() nos da numeros enteros y el Math.random() son numeros random
    numArr.push(numeroAleatorio);
}

const objectNum = {};

for (const numeros of numArr) {
    objectNum[numeros] = (objectNum[numeros] || 0) + 1;
}

console.log(objectNum);

manejo de verciones de NPM 
Ej: v2.0.4  --> Esto es una convencion
    - El 2 (major version, quiere dercir que es version con grandes cambios y puede afectar a versiones anteriores)
    - El 0 (minor version, quiere decir que hay cambios pero que no deberia de afectar a verciones anteriores)
    - El 4 (patch version son arreglos en la libreria de bugs, asi que en teoria no se deberia de estar cambiando nada)

Operadores para actualizar versiones
^2.1122.0 ("Caret") Sirve para instalar la version menor mas reciente, es decir si tenemos el caret y hacemos una actualizacion nos va a actualizar a la versioon minor mas alta
~5.0.1 ("Tilde o de negacion") solo actualiza los parches, por ejemplo si actulizamos ~5.0.1 nos va llevar a la version ~5.0.10

comandos para actualizar en NPM
comando: npm update
